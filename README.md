# swift-project-architecture

Start a new swift project with this architecture.

## 工程结构

```sh
.
└── SwiftArchitecture # 项目目录
    ├── AppDelegate.swift
    ├── Assets.xcassets # 资源文件目录
    ├── Common # 公共数据目录
    ├── Models # 实体目录
    ├── Services # 数据操作目录
    ├── Scenes # 场景目录
    │   └── Main # 具体场景目录
    │       └── MainViewController.swift
    └── Utils # 工具目录
```
